Logiciels utilisant G2S:

Boot-Repair: le "Réparateur de Démarrage"
DEBIDITOR: créer vos paquet DEBIAN de façon simple.
DeskWall: changer de fond d'écran pour Nautilus, PCManFM, Rox, Feh
GUnity: configurer votre bureau Unity pour Ubuntu 11.04
LaunchBash: création des sources d'un logiciel en bash afin de l'envoyer sur les serveurs launchpad (ppa).
Malokal: est un logiciel d'automation radio en cours de développement.
mkv_extractor_gui_v4: extraire des fichiers contenu dans des .mkv
Screencastor: enregistrer ce qu'il se passe sur votre écran
sylaba2: apprendre la prononciation des lettres et des syllabes.
Xee: un petit bot avec la librairie python-xmpp
Zenitor: Ce logiciel permet la création, rapide, complet et de façon graphique, d'un code de fenêtre Zenity
