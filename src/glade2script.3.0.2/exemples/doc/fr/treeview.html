Description

 Les données du treeview peuvent être chargées au lancement de glade2script (Voir options -t / ­­--tr )
 Elles peuvent aussi être chargées via les commandes TREE@@END, TREE@@CELL ou TREE@@INSERT.  Un fichier peut aussi être utilisé via la commande TREE@@LOAD.
 L'accès aux lignes se fait par leur numéro. L'accès à la ligne enfant s'écrit 'line:enfant' (1:0). En cas de chargement via un fichier, il est possible d'indiquer en premier élément le numéro de ligne, nécessaire pour la création d' enfant, ce numéro ne sera pas modifier en cas de sauvegarde ou récupération de données (en cas d'insertion ou de déplacement par l'user).
 L'indexation des lignes et colonnes commencent à zéro.
