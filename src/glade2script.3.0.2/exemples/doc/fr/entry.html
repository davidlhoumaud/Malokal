Description

 Zone de saisie.
 Lorsque vous utilsez des icones, si l'entry est régler sur insensible dans le glade, les icones resteront ainsi même après la commande adéquat. Pour éviter cela, griser et dégriser depuis le script.

 Les icones ont des signaux, en utilisant le callback on_entry sur ces icones, le retour dans la fonction sera different pour ces icones.
primary@TexteDeEntry ou secondary@TexteDeEntry

 Les commandes pygtk à utiliser pour la progressbarre :
progress_pulse() et set_progress_fraction(0.2)
