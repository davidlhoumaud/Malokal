Gtk.ToggleButton

Gtk.ToggleButton - Gtk.CheckButton - Gtk.RadioButton





Commandes pyGtk

Les plus utilisées: 
 set_active(is_active)
 get_active()

Documentation pyGtk: gtk.ToggleButton gtk.CheckButton gtk.RadioButton



Commandes glade2script

TOGGLE@@ACTIVE@@widget,widget
ISACTIVE@@widget



Callbacks

on_toggled
Pour agir sur les Toggles
on_active
off_active
toogle_active



Signaux

"toggled"



Description

 Ces commandes sont valables pour les CheckButton, RadioButton et ToggleButton
 
 
Pour récuperer l'état d'un check, 3 solutions:
   commande pygtk:   
echo 'GET@_check1.get_active()'
echo 'ITER@@' #si tu comptes travailler sur le résultat.
# la variable $_check1_get_active=True/False

   callback sur check: 
# renseigner signal toggled => callback on_toggled

# la fonction sera appelé à chaque action sur check avec argument True/False

   commande G2S: ISACTIVE@@


Pour activer un check:
    commande pygtk:
echo 'SET@_check1.set_active(True)'

   callback sur check:
# on_active, off_active, toggle_active, renseigner user-data avec le nom du check à modifier (à mettre sur un bouton par exemple).

   commande G2S: TOGGLE@@ACTIVE
